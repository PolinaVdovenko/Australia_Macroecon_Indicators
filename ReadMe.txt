В качестве основы для построения прогноза значений ВВП использовалась модель экономического роста для закрытой экономики, формализованная в виде рекуррентных соотношений для значений валового выпуска продукции, запаса капитала и потребления в определенный момент времени с учетом случайного фактора. Тестирование модели и калибровка её параметров производились на основе статистических данных с сайта Всемирного Банка. Одним из ключевых критериев выбора страны послужила позиция в политической и экономической мировой среде. По результатам анализа информации о макроэкономических показателях разных стран, была выбрана Австралия в период с 2002 по 2006 годы включительно как страна с достаточно стабильным периодом в экономике, подходящая для калибровки рассматриваемой модели. С технической стороны был реализован алгоритм в системе MATLAB, использующий встроенные средства для поиска оптимальных значений параметров, фигурирующих в выбранной модели. Оптимальными считались параметры, на основе которых прогноз ВВП на более длительный период строился наиболее приближенно к реальным данным.
